# ps 命令阅读指南

> 本目录包含 ps 命令的完整学习材料，从进程状态到命令参考再到实战输出。
> 按 01 → 03 的顺序阅读。

## 阅读路线

| Step | 文件 | 行数 | 关注点 |
|------|------|------|--------|
| **1** | [01_process_states.md](./01_process_states.md) | ~350 | 进程状态机（R/S/D/Z/T）、/proc/[pid]/stat 字段、D 状态排查 |
| **2** | [02_command_reference.md](./02_command_reference.md) | ~450 | ps 参数分类、格式字段速查表、10+ 场景命令 |
| **3** | [03_command_output_demo.md](./03_command_output_demo.md) | ~550 | 实际执行命令并逐列注解、STAT 字段解读 |

## 阅读建议

1. **先通读 Step 1**：理解进程状态机 — 这是解读 STAT 字段和排查问题的基础
2. **Step 2 是工具书**：按参数分类速查，遇到不确定的选项时回来查
3. **Step 3 对照执行**：在自己的机器上跑一遍，加深对每列含义的理解
