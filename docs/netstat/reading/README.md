# netstat + ss 阅读指南

> 本目录包含 netstat/ss 的深入学习材料，涵盖 TCP 状态机、工具对比、命令参考与实战解析。
> 按 01 → 04 的顺序阅读。

## 阅读路线

| 序号 | 文件 | 内容 | 关注点 |
|------|------|------|--------|
| **01** | [01_tcp_state_machine.md](./01_tcp_state_machine.md) | TCP 状态机详解 | 11 个状态、三次握手/四次挥手、状态转换图 |
| **02** | [02_netstat_vs_ss.md](./02_netstat_vs_ss.md) | netstat 与 ss 对比 | /proc/net/tcp vs netlink、性能差异、数据源原理 |

## 阅读建议

1. **先读 01（TCP 状态机）**：这是理解 ss/netstat 输出的基础 — 每个连接的 State 列就是 TCP 状态
2. **再读 02（工具对比）**：理解两代工具的原理差异，明白为什么 ss 更快
3. **然后 03（命令参考）**：掌握常用参数，遇到具体场景可回查
4. **最后 04（输出实战）**：对照真实输出逐行理解，结合自己的系统练习

## 文件说明

- 01-02 为概念性文档，帮助建立知识框架
- 03-04 为实操文档，所有命令输出均来自真实系统
- QA.md 为问答集，记录学习中遇到的问题
