# netstat + ss 阅读指南

> 本目录包含 netstat/ss 的深入学习材料，涵盖 TCP 状态机、命令参考与实战解析。
> 按 01 → 03 的顺序阅读。

## 阅读路线

| 序号 | 文件 | 内容 | 关注点 |
|------|------|------|--------|
| **01** | [01_tcp_state_machine.md](./01_tcp_state_machine.md) | TCP 状态机详解 | 11 个状态、三次握手/四次挥手、状态转换图 |
| **02** | [02_command_reference.md](./02_command_reference.md) | 完整命令参考 | netstat/ss 全部参数、10+ 场景命令组合、数据源原理 |

## 阅读建议

1. **先读 01（TCP 状态机）**：这是理解 ss/netstat 输出的基础 — 每个连接的 State 列就是 TCP 状态
2. **再读 02（命令参考）**：掌握 netstat/ss 的完整参数和数据源原理（/proc/net/tcp vs netlink），遇到具体场景可回查
3. **最后 03（输出实战）**：对照真实输出逐行理解，结合自己的系统练习
